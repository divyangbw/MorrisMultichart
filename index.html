What is Multicharts?

An extension for Morris charts, allowing users to display multiple charts from the same data source.

A Simple Example

Setup

						
// To get all data options, see Morris.js project page
var options = {
	element: 'example-chart',
	data: [
		{ y: '2006', a: 100, b: 90 },
		{ y: '2007', a: 75,  b: 65 },
		{ y: '2008', a: 50,  b: 40 },
		{ y: '2009', a: 75,  b: 65 },
		{ y: '2010', a: 50,  b: 40 },
		{ y: '2011', a: 75,  b: 65 },
		{ y: '2012', a: 100, b: 90 }
	],
	xkey: 'y',
	ykeys: ['a', 'b'],
	labels: ['Series A', 'Series B']
};

var exampleChartOne = MorrisMultiCharts;
exampleChartOne.InitializeGraphs(options);

// If you decide to use only one instance of the chart on the page, 
// you can infact call multicharts like this:
MorrisMultiCharts.InitializeGraphs(options);

Required Fields

  1. element, which is the id of the element to convert to charts. Example: For a div, id="example-chart".
  2. data
  3. xkey: What represents the x-key from the data
  4. ykeys: what represents the y-keys from the data
  5. labels: Although this wont break the charts if you don't define it.
